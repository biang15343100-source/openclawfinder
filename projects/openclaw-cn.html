OpenClaw-CN

A community-maintained Chinese distribution of OpenClaw with Feishu, DingTalk, WeCom integrations, DeepSeek/Qwen defaults, and domestic mirrors.

Description

OpenClaw-CN exists because a lot of the friction of running OpenClaw in China is not about the runtime itself. It is about defaults: which messaging channels come wired up, which model providers are reachable, whether docs and dashboard strings are in Chinese, and whether npm mirrors are fast enough.

This variant rewires all of those defaults in one place instead of asking each user to patch them locally. It tracks upstream OpenClaw and adds localized CLI and dashboard strings, Feishu/DingTalk/WeCom integrations, and first-class support for domestic models and registry mirrors.

  • Deep CLI and dashboard localization, tracking upstream OpenClaw releases.
  • Built-in integrations for DingTalk, WeCom, Feishu, QQ, and WeChat out of the box.
  • First-class DeepSeek and Qwen presets, with npmmirror and Aliyun registry support.
