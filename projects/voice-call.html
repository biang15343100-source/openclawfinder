OpenClaw voice plugin

Voice Call

An OpenClaw plugin that gives agents phone-calling abilities through providers such as Twilio, Telnyx, and Plivo.

Description

Voice Call is the part of the ecosystem that turns an OpenClaw agent into something that can actually place or receive phone calls. Public docs describe support for multiple providers, outbound and inbound modes, webhook security controls, streaming, and tool actions for continuing, speaking, ending, and checking the status of a call.

Its practical value is obvious: once agents can call people, use cases shift from chat-only assistants into reminders, customer support, verification flows, and hybrid workflows. It becomes especially powerful when combined with tools like Lobster, where a phone call can be one audited step inside a larger human-in-the-loop process.

  • Supports telephony providers such as Twilio, Telnyx, Plivo, and mock mode.
  • Covers both outbound calls and inbound-call handling policies.
  • Extends OpenClaw into real-world voice and phone workflows.
