Official ecosystem tool

ClawHub

The official marketplace and registry for discovering, installing, and publishing OpenClaw skills.

Description

ClawHub is part of the official tooling layer around OpenClaw. It is where skills can be found, installed, and shared, which makes the ecosystem feel bigger than one runtime and one docs site.

That matters because healthy platforms need extension points, not only a core app. ClawHub is one of the clearest signals that OpenClaw is trying to support a broader tool and skill marketplace rather than staying a closed runtime surface.

  • Documented in the official OpenClaw docs.
  • Backed by a public GitHub repository.
  • A strong fit for a separate ecosystem-tools category in the directory.
