Zero-setup desktop installer

OneClaw

A packaged OpenClaw desktop client with a bundled Node runtime, multi-provider key management, and one-minute install on macOS and Windows.

Description

OneClaw targets the gap between curious users and a working OpenClaw install. Instead of asking people to set up Node, npm, and environment variables, it ships a signed desktop app that bundles the runtime and exposes a clean settings panel for API keys and providers.

Positioning-wise it sits between the pure runtime forks and the productized research consoles. It does not try to change what OpenClaw is. It just removes the installation tax, which is one of the most common reasons people bounce off the ecosystem on their first try.

  • Signed desktop app for macOS (Intel/Apple Silicon) and Windows (x64/ARM64) with auto-updates.
  • Bundled Node runtime so there is no separate Node or npm install required.
  • Local API key storage and mid-chat model switching across Anthropic, OpenAI, Google, and Kimi.
